# ETL-sales-pipeline
This is a beginner-friendly ETL (Extract–Transform–Load) pipeline built in Python using CSV files as the data source.
Sales ETL Project

Description
This project demonstrates an ETL (Extract, Transform, Load) pipeline for processing sales data. The pipeline extracts raw sales data from CSV files, transforms it to clean and structured formats, and loads it into a database. The project is designed for learning ETL processes.

Features
 Extract data from CSV files 
 Clean and transform raw sales data (e.g., remove duplicates, normalize fields)  
 Aggregate sales metrics
 Load transformed data into a database (e.g., SQLite)  
 Modular design for easy extension  

Technologies Used
 Python 3.x  
 Pandas(for data manipulation)  
 SQLite (as the target database)

   git clone (https://github.com/Chandira77/ETL-sales-pipeline.git)

2. Navigate to the project folder:
   cd sales-etl

3. Install dependencies:
